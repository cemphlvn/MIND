MIND

Specification

Scope

This document specifies the observable behavior and invariants of the MIND runtime. It is intentionally implementation-agnostic.

Core entities

runtime → execution context
state   → persistent cognitive memory
hint    → compressed recall output

Constants

CR_EPSILON        = 0.05   // plasticity floor
CR_SIM_THRESHOLD  = 0.85   // reinforcement threshold
CR_DECAY_RATE     = 0.995  // crystallization rate
CR_RECOVERY_RATE  = 1.0005 // openness recovery

Plasticity

Plasticity ∈ (ε, 1]. It measures malleability. Plasticity decays only through reinforcement. Plasticity never reaches zero.

Confidence

confidence = similarity × stability × weight_factor

stability = 1 - plasticity
weight_factor = weight / (weight + 1)

Confidence is derived, never asserted.

Determinism

Given identical inputs, ordering, and delta values, the system must evolve identically across machines.

Non-goals

This specification excludes text generation, tokenization, probabilistic randomness, and gradient-based learning.
